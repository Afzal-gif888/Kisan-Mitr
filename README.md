# Kisan Mitr - Smart Farming Assistant

Kisan Mitr is a production-ready agricultural intelligence platform designed to help farmers in Andhra Pradesh make informed decisions.

## Features
- **Weather Intelligence**: Real-time insights and seasonal predictions.
- **Crop Selector**: Soil and weather-based crop recommendations.
- **Market Prices**: Live commodity price data for Andhra Pradesh.
- **Farming Guides**: Detailed localized guides for major crops.

## Development
```bash
npm install
npm run dev
```

## Production
```bash
npm run build
npm run preview
```
